3 Deployment guide — host it yourself in 15 minutes

Once deployed, you share a URL — never a file. You control access. You can revoke it instantly.
  • 1
    Create a free GitHub account Go to github.com → Sign up (free). Use your personal email, not a work email.
  • 2
    Create a new repository named "nexusos" Click "New repository" → name it "nexusos" → set to Private → click Create.
  • 3
    Upload both files: index.html and app.html Click "Add file" → "Upload files" → drag both HTML files → commit changes.
  • 4
    Create a free Netlify account at netlify.com Sign up with GitHub → click "Import from Git" → select your nexusos repo.
  • 5
    Deploy to Netlify — your site goes live at a URL you control Netlify auto-deploys. You get a URL like nexusos-ga.netlify.app — share this URL with clients, never the file.
  • 6
    Change credentials to revoke access instantly Open app.html → find the USERS object → change the password for that client → commit to GitHub → Netlify auto-updates in 30 seconds. Client is locked out instantly.
    const USERS = { "ceo-mc2026": { pass: "HomeCare@CEO26!", role: "CEO", name: "Marcus" }, "ea-giovanni": { pass: "NexusEA@GA2026!", role: "EA", name: "Giovanni" } }; // To revoke: change or delete the client's entry above
  • 7
    Add a custom domain (optional, ~$12/year) Buy nexusos.app or nexusos.io from Namecheap (~$12/yr). Connect to Netlify in Settings → Domain. Now your URL is nexusos.app — professional and completely yours.
  • 8
    What to say when you share it with the CEO "Here are your NexusOS credentials. Access is via the link below — it's a secure hosted platform I manage. Your login is personal to you and active while we're working together."
