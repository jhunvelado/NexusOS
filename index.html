Built for security-conscious executives

HIPAA-aligned architecture from the ground up.

⏱️

15-Minute Session Timeout

Automatic logout after 15 minutes of inactivity with a 60-second warning toast.

📋

Complete Audit Trail

Every login, logout, screen access, and payment logged to an RLS-protected audit_logs table.

🔐

Row-Level Security (RLS)

Supabase RLS ensures each workspace sees only its own data. No shared tables, no cross-org leaks.

🚫

No Secrets in Frontend

Payment secret keys live exclusively in Supabase Edge Functions. Only public keys in client-side code.

🔒

Encrypted at Rest

All data in Supabase PostgreSQL with AES-256 encryption at rest and TLS in transit.

🛡️

Supabase Auth

JWT-based authentication with email+password and Google OAuth. Tokens rotate automatically.

Note on HIPAA BAA: GitHub Pages cannot sign a BAA. For fully HIPAA-covered deployments, host on a provider that offers a BAA (e.g., Vercel Team/Enterprise, AWS). Supabase BAA is available on their Team and Enterprise plans.
