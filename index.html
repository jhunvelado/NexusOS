Security & Compliance

Enterprise-grade security,
built in from day one

NexusOS is architected for healthcare and PE environments where data security isn't optional.

15-Minute Session Timeout

Auto-logout after 15 minutes of inactivity with a 60-second warning. No unattended session ever stays open. HIPAA-aligned inactivity control.

Full Audit Logging

Every login, logout, data access, record create/update/delete, and feature gate hit is written to an immutable audit log with user ID, timestamp, and action type.

Row Level Security (RLS)

Every query is filtered by workspace. No cross-tenant data leakage is possible at the database level. Enforced by Supabase Postgres policies — not application code.

No Secrets in Frontend

API keys and payment secrets live only in Supabase Edge Functions. The client-side code contains zero credentials that could be extracted from the browser.

Encrypted at Rest & Transit

All data stored in Supabase is AES-256 encrypted at rest. All traffic uses TLS 1.3. Data never moves in plaintext.

Auth via Supabase

Email + password with confirmed email flow. JWT-based sessions. Password reset via secure email link. No shared credentials between users.

Note on HIPAA: NexusOS is built with HIPAA-aligned controls. To achieve full HIPAA compliance, operators must sign a Business Associate Agreement with Supabase (Team/Enterprise plan) and host the frontend on a HIPAA-eligible provider. Contact us for guidance.
